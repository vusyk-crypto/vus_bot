Чтобы запустить виртуальное окружение в Windows, 
нужно сделать следующее:

Открой командную строку (Cmd) 
или терминал PowerShell.

Перейди в папку с виртуальным окружением:

cd C:\Users\User\vus_bot

Активируй виртуальное окружен
vus_bot\Scripts\activate

Чтобы залить изменения в ГитХаб

Проверь статус файлов: Чтобы увидеть изменения, 
которые ты сделал, выполни команду:


git status

Добавь измененный файл в отслеживание: 
Если файл index.html изменён, добавь его:


git add index.html   или git add .

Закоммить изменения: Коммит – это сохранение изменений 
в локальном репозитории. Введи описание изменений:


git commit -m "Обновление кастомной страницы: Изменение пропорции лого и синей подложки с помощью О1"

Отправь изменения на GitHub: Отправь 
изменения в удаленный репозиторий:


git push origin main
